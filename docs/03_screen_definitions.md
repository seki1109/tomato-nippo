## SCR-01 ログイン画面

**概要:** メールアドレスとパスワードで認証する。

**レイアウト:**

```
┌──────────────────────────────────┐
│         営業日報システム           │
│                                  │
│  メールアドレス                    │
│  [________________________]      │
│                                  │
│  パスワード                        │
│  [________________________]      │
│                                  │
│         [ログイン]                 │
└──────────────────────────────────┘
```

**入力項目:**

| 項目名 | 型 | 必須 | バリデーション |
|---|---|---|---|
| メールアドレス | text | ○ | メール形式 |
| パスワード | password | ○ | 8文字以上 |

**ボタン:**

| ボタン名 | アクション |
|---|---|
| ログイン | 認証処理 → 成功時: SCR-02へ遷移 / 失敗時: エラーメッセージ表示 |

## SCR-02 日報一覧画面

**概要:** ログインユーザーの日報を日付降順で一覧表示する。上長は全営業の日報を閲覧可能。

**レイアウト:**

```
┌──────────────────────────────────────────────────────┐
│ 営業日報システム              [田中 太郎▼] [ログアウト] │
├──────────────────────────────────────────────────────┤
│ 日報一覧                           [+ 新規作成]       │
│                                                      │
│ 絞り込み: [担当者▼] [年月▼] [ステータス▼] [検索]      │
│                                                      │
│ ┌──────────────────────────────────────────────────┐ │
│ │ 日付       │ 担当者   │ ステータス │ 更新日時    │ │
│ ├──────────────────────────────────────────────────┤ │
│ │ 2026-03-27 │ 田中太郎 │ 提出済み   │ 18:30      │ │
│ │ 2026-03-26 │ 田中太郎 │ 下書き     │ 17:45      │ │
│ │ ...        │ ...      │ ...        │ ...        │ │
│ └──────────────────────────────────────────────────┘ │
│                      [< 前へ] 1/5 [次へ >]           │
└──────────────────────────────────────────────────────┘
```

**表示項目:**

| 項目名 | 説明 |
|---|---|
| 日付 | 日報の対象日 |
| 担当者 | 日報作成者（上長ロールのみ表示） |
| ステータス | 下書き / 提出済み |
| 更新日時 | 最終更新日時 |

**操作:**

| 操作 | アクション |
|---|---|
| 行クリック | SCR-04（日報詳細）へ遷移 |
| 新規作成ボタン | SCR-03（日報作成）へ遷移 |
| 絞り込み検索 | 条件に合う日報を再表示 |

## SCR-03 日報作成画面

**概要:** 新規日報を作成する。訪問記録を複数行追加できる。

**レイアウト:**

```
┌──────────────────────────────────────────────────────┐
│ 営業日報システム              [田中 太郎▼] [ログアウト] │
├──────────────────────────────────────────────────────┤
│ 日報作成                                              │
│                                                      │
│ 日付: [2026-03-27        ]                           │
│                                                      │
│ ── 訪問記録 ──────────────────────────────────────── │
│ ┌────────────────────────────────────────────────┐   │
│ │ # │ 顧客名           │ 訪問内容         │ 操作 │   │
│ ├────────────────────────────────────────────────┤   │
│ │ 1 │ [株式会社〇〇 ▼] │ [____________] │ [削除]│   │
│ │ 2 │ [株式会社△△ ▼] │ [____________] │ [削除]│   │
│ └────────────────────────────────────────────────┘   │
│                          [+ 訪問先を追加]             │
│                                                      │
│ ── 課題・相談（Problem） ─────────────────────────── │
│ [                                                  ] │
│ [                                                  ] │
│                                                      │
│ ── 明日やること（Plan） ──────────────────────────── │
│ [                                                  ] │
│ [                                                  ] │
│                                                      │
│          [下書き保存]          [提出する]             │
└──────────────────────────────────────────────────────┘
```

**入力項目:**

| 項目名 | 型 | 必須 | バリデーション |
|---|---|---|---|
| 日付 | date | ○ | 今日以前の日付、同日の日報が未存在であること |
| 顧客名（訪問記録） | select | ○ | 顧客マスターから選択 |
| 訪問内容（訪問記録） | textarea | ○ | 1000文字以内 |
| 課題・相談（Problem） | textarea | | 2000文字以内 |
| 明日やること（Plan） | textarea | | 2000文字以内 |

**ボタン:**

| ボタン名 | アクション |
|---|---|
| 訪問先を追加 | 訪問記録行を1行追加 |
| 削除 | 対象行の訪問記録を削除 |
| 下書き保存 | ステータス DRAFT で保存 → SCR-04へ遷移 |
| 提出する | ステータス SUBMITTED で保存 → SCR-04へ遷移 |

## SCR-04 日報詳細・編集画面

**概要:** 日報の詳細表示と編集、上長によるコメント投稿を行う。

**レイアウト:**

```
┌──────────────────────────────────────────────────────┐
│ 営業日報システム              [田中 太郎▼] [ログアウト] │
├──────────────────────────────────────────────────────┤
│ 日報詳細 - 2026-03-27（田中 太郎）  [ステータス: 提出済み]│
│                                    [編集] [削除]      │
│                                                      │
│ ── 訪問記録 ──────────────────────────────────────── │
│ ┌────────────────────────────────────────────────┐   │
│ │ # │ 顧客名      │ 訪問内容                      │   │
│ ├────────────────────────────────────────────────┤   │
│ │ 1 │ 株式会社〇〇 │ 新製品の提案を実施。前向きな反応。│  │
│ │ 2 │ 株式会社△△ │ 契約更新の打ち合わせ。来週返答。 │  │
│ └────────────────────────────────────────────────┘   │
│                                                      │
│ ── 課題・相談（Problem） ─────────────────────────── │
│ 株式会社△△の担当者が変わり、関係構築が必要。          │
│                                                      │
│ ── 明日やること（Plan） ──────────────────────────── │
│ 株式会社〇〇へ提案資料を送付する。                     │
│                                                      │
│ ── コメント ──────────────────────────────────────── │
│ ┌────────────────────────────────────────────────┐   │
│ │ 山田部長 | 2026-03-27 19:00          [編集][削除]│   │
│ │ △△社の件、来週フォローアップ頼みます。            │   │
│ └────────────────────────────────────────────────┘   │
│                                                      │
│ コメントを追加（上長のみ）:                            │
│ [                                                  ] │
│                              [コメントを投稿]         │
└──────────────────────────────────────────────────────┘
```

**表示項目:**

| 項目名 | 説明 |
|---|---|
| 日付・担当者・ステータス | 日報ヘッダー情報 |
| 訪問記録一覧 | 顧客名・訪問内容をリスト表示 |
| Problem | 今の課題・相談 |
| Plan | 明日やること |
| コメント一覧 | 上長のコメント（投稿者・日時・本文） |

**ボタン:**

| ボタン名 | 表示条件 | アクション |
|---|---|---|
| 編集 | 作成者本人 かつ DRAFT のみ | 編集モードへ切り替え |
| 削除 | 作成者本人 かつ DRAFT のみ | 日報削除 → SCR-02へ遷移 |
| コメントを投稿 | MANAGER ロールのみ | コメントを保存して画面を更新 |
| コメント編集 | 自分のコメントのみ | インライン編集モードへ |
| コメント削除 | 自分のコメントのみ | コメントを削除 |

| 編集 | SCR-06（編集）へ遷移 |

## SCR-06 顧客マスター登録・編集画面

**概要:** 顧客の新規登録または既存顧客の情報編集を行う。

**入力項目:**

| 項目名 | 型 | 必須 | バリデーション |
|---|---|---|---|
| 会社名 | text | ○ | 200文字以内 |
| 担当者名 | text | | 100文字以内 |
| 電話番号 | text | | 電話番号形式 |
| 住所 | text | | 500文字以内 |

**ボタン:**

| ボタン名 | アクション |
|---|---|
| 保存 | 入力内容を保存 → SCR-05へ遷移 |
| キャンセル | 変更を破棄 → SCR-05へ遷移 |

## SCR-08 ユーザーマスター登録・編集画面

|---|---|---|---|
| 氏名 | text | ○ | 100文字以内 |
| メールアドレス | text | ○ | メール形式・重複不可 |
| パスワード | password | ○（新規のみ） | 8文字以上、英数字混在 |
| ロール | select | ○ | SALES / MANAGER / ADMIN |
| 部署名 | text | | 100文字以内 |
| 有効フラグ | toggle | ○ | |

**ボタン:**

| ボタン名 | アクション |
|---|---|
| 保存 | 入力内容を保存 → SCR-07へ遷移 |
| キャンセル | 変更を破棄 → SCR-07へ遷移 |
