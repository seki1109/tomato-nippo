## 1. システム概要

営業担当者が日々の顧客訪問内容・課題・翌日計画を記録し、上長がコメントを通じてフォローアップできる営業日報管理システム。

### 目的

- 営業活動の可視化・記録
- 上長による適切なフォローアップ・指導
- 顧客訪問履歴の蓄積

---

## 2. 利用者・ロール定義

| ロール | 説明 | 主な操作 |
|---|---|---|
| **SALES**（営業） | 日報を作成・編集する担当者 | 日報作成、訪問記録入力、Problem/Plan記入 |
| **MANAGER**（上長） | 日報にコメントする管理職 | 日報閲覧、コメント投稿・編集・削除 |
| **ADMIN**（管理者） | マスターデータを管理する担当者 | 顧客マスター管理、ユーザーマスター管理 |

## 3. 機能要件

### 3-1. 日報管理

| # | 機能 | 説明 |
|---|---|---|
| F01 | 日報作成 | 営業が日付・訪問記録・Problem・Planを入力して日報を作成する |
| F02 | 日報編集 | 提出前の日報を編集できる |
| F03 | 日報提出 | 作成した日報を上長に提出する（ステータス: DRAFT → SUBMITTED） |
| F04 | 日報一覧 | 自分の日報一覧を日付降順で表示する |
| F05 | 日報詳細 | 日報の全内容（訪問記録・Problem・Plan・コメント）を表示する |

### 3-2. 訪問記録管理

| # | 機能 | 説明 |
|---|---|---|
| F06 | 訪問記録追加 | 日報に顧客と訪問内容を複数行追加できる |
| F07 | 訪問記録編集 | 訪問記録の顧客・内容を変更できる |
| F08 | 訪問記録削除 | 訪問記録を行単位で削除できる |
| F09 | 訪問順序変更 | 訪問した順序を並び替えられる |

### 3-3. コメント管理

| # | 機能 | 説明 |
|---|---|---|
| F10 | コメント投稿 | 上長が日報に対してコメントを投稿できる |
| F11 | コメント編集 | 自分が投稿したコメントを編集できる |
| F12 | コメント削除 | 自分が投稿したコメントを削除できる |

### 3-4. マスター管理

| # | 機能 | 説明 |
|---|---|---|
| F13 | 顧客マスター管理 | 顧客の登録・編集・削除 |
| F14 | ユーザーマスター管理 | ユーザーの登録・編集・ロール変更・無効化 |

### 3-5. 認証

| # | 機能 | 説明 |
|---|---|---|
| F15 | ログイン | メールアドレスとパスワードで認証 |
| F16 | ログアウト | セッションを破棄してログイン画面へ戻る |
